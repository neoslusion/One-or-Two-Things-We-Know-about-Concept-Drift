\chapter*{ABSTRACT}
\addcontentsline{toc}{chapter}{ABSTRACT}

Currently, concept drift is one of the major challenges that degrades the performance of machine learning systems, especially in continuously changing streaming data environments. This thesis focuses on unsupervised drift detection and classification, with two main contributions: (1) integrating \textbf{IDW-MMD} (Inverse Density-Weighted MMD) into ShapeDD, achieving a 16.5$\times$ throughput improvement over the original method while maintaining zero false positives on the detection benchmark; and (2) proposing \textbf{SE-CDT} (ShapeDD-Enhanced Concept Drift Type), an unsupervised drift type classification method based on MMD signal shape analysis, achieving 77.4\% category accuracy (TCD vs PCD). Additionally, the thesis implements a \textbf{Type-Specific Adaptation} framework that selects appropriate model update strategies based on the classified drift type, improving accuracy by +22.14\% over no adaptation on Stepping Drift scenarios. The system is designed on an Apache Kafka architecture for real-time streaming data processing.
