\chapter*{ABSTRACT}
\addcontentsline{toc}{chapter}{ABSTRACT}

Concept drift remains one of the major challenges that degrade the performance of machine learning systems, especially in continuously changing streaming environments. This thesis focuses on unsupervised drift detection and classification.

The thesis makes two main contributions. First, it integrates \textbf{IDW-MMD} (Inverse Density-Weighted MMD) into ShapeDD as the validation step of a hybrid detector (Standard MMD trace combined with IDW under a Gamma-approximation null). The resulting test is properly calibrated (empirical Type-I error 0.04--0.10, within the 95\% confidence interval of the nominal $\alpha = 0.05$) and reaches a competitive F1-score of 0.454 on a 14-dataset detection benchmark with $\delta = 75$ tolerance, while running at roughly the same end-to-end runtime as the original permutation-based ShapeDD (1.94 vs.\ 1.99 seconds per 10{,}000-sample stream). Second, it proposes \textbf{SE-CDT} (ShapeDD-Enhanced Concept Drift Type), an unsupervised drift type classification method based on MMD signal shape analysis, achieving 82.2\% category accuracy (TCD vs.\ PCD) and 65.8\% micro-averaged / 55.7\% macro-averaged subcategory accuracy across five drift types. SE-CDT is augmented with a \textit{Concept Memory} module that pushes Recurrent-drift identification from 33.1\% to 91.5\% and an \textit{online self-calibration} mechanism that adaptively adjusts classification thresholds at runtime.

Additionally, the thesis implements a \textbf{Type-Specific Adaptation} framework that selects an appropriate model-update strategy based on the classified drift type, improving accuracy by $+16.57$pp on Stepping Drift and $+17.10$pp on Sudden Drift over a no-adaptation baseline (Friedman test, $N = 30$ runs, $p < 10^{-15}$ on all three scenarios). The system is deployed on an Apache Kafka architecture suitable for real-time streaming pipelines.
